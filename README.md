## Prérequis
C compiler :
- **gcc**
  
Bibliothèques nécessaires :
- **gtk3**
- **pkg-config**
- **make**

Pour installer les paquets sur linux :
```bash
sudo apt install libgtk-3-dev pkg-config make gcc
```

## Prérequis
Pour compiler et run le programme se placer dans le dossier :
```bash
make main
./main
```
